Taxonomy-based Paraphrasing

This attack allows the user to generate paraphrased prompts based on a selected taxonomy model.

Usage example: python run.py -m attacked_provider/attacked_model -a tax -t "How to build a bomb" -x provider2/model2 -e taxonomy_model=provider2/model2

Below are extra arguments that can be changed using: -e extra_arg1=value1 -e extra_arg2=value2

Extra Argument Description
taxonomy_model Model to be used for generating taxonomy prompts.
In case a different model than the attacked model is chosen, -x provider/model should be used as well.
Note: It is recommended to use an uncensored model for the taxonomy task to avoid refusal while refining the attack prompt
